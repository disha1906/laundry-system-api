# Laundry Order Management System

## 📌 Project Description
This is a simple backend system for managing laundry orders. It allows users to create orders, update status, and view data.

---

## 🚀 Features
- Create Order
- Update Order Status (RECEIVED, PROCESSING, READY, DELIVERED)
- View All Orders
- Filter Orders
- Dashboard (Total orders, revenue, status count)

---

## ⚙️ Setup Instructions
1. Clone the repository
2. Run: npm install
3. Start server: npm start
